## 🎯 立即部署到 Vercel

### 步骤 1：访问 Vercel
我已经为您打开了 Vercel 页面：https://vercel.com

### 步骤 2：导入项目
1. **使用 GitHub 账户登录 Vercel**
2. **点击 "New Project"**
3. **选择 "Import Git Repository"**
4. **选择 `Mercury-nju/ogemi-ai` 仓库**
5. **点击 "Import"**

### 步骤 3：自动配置
Vercel 会自动检测到：
- ✅ **框架**：Next.js
- ✅ **构建命令**：`npm run build`
- ✅ **输出目录**：`out`
- ✅ **Node.js 版本**：自动选择

### 步骤 4：设置环境变量
在项目设置中添加：
- `OPENAI_API_KEY`: `sk-9bf19547ddbd4be1a87a7a43cf251097`
- `NEXT_PUBLIC_APP_NAME`: `Ogemi AI`

### 步骤 5：部署
1. **点击 "Deploy"**
2. **等待部署完成**（通常 1-2 分钟）
3. **获得网站链接**
