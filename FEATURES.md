# Wula.ai 功能特性详解

## 🎯 核心功能

### 1. 图片转视频生成
- **AI 驱动**: 使用通义千问 API 进行智能图像理解
- **自然语言提示**: 用户只需描述想要的效果
- **快速生成**: 秒级生成专业级视频
- **多格式支持**: JPG、PNG、WEBP
- **高质量输出**: 支持 HD/4K 导出

### 2. 动画预设系统
- 🚶 **Walking**: 人物行走动画
- 🔍 **Zoom In**: 镜头缩放效果
- 📹 **Pan**: 左右平移镜头

- 💨 **Wind Effect**: 风吹效果
- ☁️ **Moving Clouds**: 云层飘动

## 📊 数据流

```
用户上传图片
    ↓
Base64 编码
    ↓
发送到 API (/api/generate-video)
    ↓
调用通义千问 API
    ↓
生成视频描述
    ↓
模拟视频生成
    ↓
返回视频 URL
    ↓
显示在预览区
    ↓
用户下载/分享
```
